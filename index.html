The problem

The list repeats. The work should not.

Weekly shopping often starts with the same chore: remember which meals are coming up, rewrite their ingredients, add the usual basics, then clean up duplicates by hand. Wocheneinkauf keeps that work reusable.

01

Recipes get typed again

Save recipe ingredients once, including amounts, and reuse them whenever the meal returns.

02

Basics are easy to forget

Keep recurring staples in the catalog and add them beside the recipes for this shop.

03

Shared lists drift

Use one iCloud-backed household library instead of passing separate notes around.

How it works
